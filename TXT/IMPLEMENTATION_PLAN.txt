KEY TECHNICAL DECISIONS (for AI agent reference)
────────────────────────────────────────────────────────────────────────────────

  Decision                Rationale
  ──────────────────────  ──────────────────────────────────────────────────
  Next.js App Router      API routes + frontend in one repo. Middleware handles
                          auth cleanly.
  Supabase                Built-in Realtime, Auth, and Postgres. No Socket.io.
  Groq (not OpenAI)       Llama 3.3 70B is fast and cheap for narration.
  Server-side dice        Client dice can be cheated. All rolls are server-auth.
  XML event tags          More reliable than regex on prose. Claude rarely
                          breaks XML when instructed clearly.
  Layered memory          Full campaign history exceeds context limits.
                          Summarization keeps costs and tokens manageable.
  Claude Haiku (copilot)  10x cheaper than Sonnet. Quick suggestions don't
                          need Sonnet's reasoning power.
  Zustand (not Redux)     Simpler for game state. Works cleanly with Supabase
                          Realtime subscriptions.
  Dark mode only          Hardcoded 'dark' class on <html>. ThemeProvider
                          removed. Light mode breaks the atmosphere.
